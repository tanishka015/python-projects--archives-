# Employee Salary Calculator 💼

## Description
The Employee Salary Calculator is a Python program that calculates an employee's total salary based on basic salary, allowances, and deductions.

This project demonstrates how to perform calculations using user input and apply business logic to compute net salary.

---

## Features

- Accepts employee details
- Calculates allowances (e.g., HRA, Bonus)
- Calculates deductions (e.g., Tax)
- Displays total and net salary
- Simple command-line interface

---

## How to Run

1. Make sure Python is installed on your system
2. Navigate to the project folder
3. Run the following command:

```bash
python main.py
```

---

## Example Calculation

Basic Salary: 20000  
HRA (20%): 4000  
Bonus (10%): 2000  
Tax (5%): 1000  

Net Salary = 25000
